# Git Rebase Commands

```
$ git remote -v
$
$ git remote add upstream https://YourUsername@Bitbucket.YourCompany.com:Port/YourUsername/YourRepository.git
$
$ git remote -v
$
$ git fetch upstream
$
$ git remote remove upstream
```

```
$ git fetch origin
$
$ git rebase origin/master 
$
$ git push
```
